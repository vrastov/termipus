# Модуль `extension`

Браузерное расширение Termipus для Chrome и Firefox. Инжектирует кнопку открытия терминала в
интерфейсы GitLab/GitHub и отображает xterm.js панель.

## Структура `src/`

- `content/` — скрипты содержимого: поиск DOM-элементов GitLab/GitHub и добавление кнопки
- `terminal/` — компонент терминала на базе xterm.js, подключение к WebSocket
- `popup/` — UI для настроек расширения (адрес сервера, тема)
- `background/` — service worker для управления состоянием и коммуникации

## Сборка

Установите зависимости:

```bash
npm install
```

Сборка для Chrome:

```bash
npm run build:chrome
```

Сборка для Firefox:

```bash
npm run build:firefox
```

Артефакты собираются в папку `dist/`.

## Настройка адреса сервера

Расширение не использует `.env`-файлы. Адрес WebSocket-сервера хранится в `chrome.storage.sync`
и настраивается через интерфейс Popup. По умолчанию используется `ws://localhost:8080/terminal`.

Пользователь может изменить адрес в любой момент — новое значение сохраняется и применяется для
последующих подключений.
